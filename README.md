# Zaprosi (LINQ queries)

Проект для работы с LINQ-запросами на C#.

## Что делает программа

**Создаёт списки:**
* **Materials** (код, название, возрастная группа)
* **Toys** (название, код материала, дата выпуска)

**Выполняет LINQ-запросы:**
* **a)** Все игрушки (выборка объектов)
* **b)** Сортировка по названию (убывание)
* **c)** Сортировка по названию и дате выпуска
* **d)** Фильтр: игрушки, выпущенные до июня
* **e)** Фильтр: игрушки, выпущенные с мая по август
* **f) Join:** объединение данных из двух списков (игрушка + материал) через **анонимные типы** (`select new`)

## Результат
Программа выводит данные в консоль и сохраняет полный отчет в файл `zaprosi_output.txt`.

## Как запустить
1. Открыть проект в Visual Studio.
2. Запустить `Program.cs` (F5).
3. Файл с результатом появится в папке с программой.

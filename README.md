# Zaprosi (LINQ queries)

Проект для работы с LINQ-запросами на C#.

## Что делает программа

1. Создаёт списки:
   - **Материалы** (код, название, возрастная группа)
   - **Игрушки** (название, код материала, месяц выпуска)

2. Выполняет LINQ-запросы:
   - Все игрушки
   - Сортировка по названию (убывание)
   - Сортировка по названию и дате
   - Фильтр: игрушки до июня
   - Фильтр: игрушки с мая по август

3. Сохраняет результат в файл `zaprosi_output.txt`

## Как запустить

1. Открыть проект в Visual Studio
2. Запустить `Program.cs` (F5)
3. Результат появится в папке `bin/Debug/zaprosi_output.txt`
